# House Price Prediction

Machine learning project predicting house sale prices.

## Methods
- Data cleaning: missing value imputation, one-hot encoding
- Log transformation to correct price skew
- Models compared: Linear, Ridge, Lasso, ElasticNet
- Hyperparameter tuning with GridSearchCV
- VIF analysis for multicollinearity detection

## Key Results
- Lasso used for automatic feature selection
- ElasticNet tuned for optimal L1/L2 ratio
- Evaluated with R², RMSE, residual plots

## Tools
Python, pandas, scikit-learn, matplotlib, seaborn
